# Expense Tracker

Expense Tracker helps you know how much to spend.

## Installation

Use the vcs [git](https://git.scm.com/) to install expense-tracker-react-native
.

```bash
git clone expense-tracker-react-native

```

## Usage

```javascript
cd expense-tracker-react-native

npm install

npm start
```
